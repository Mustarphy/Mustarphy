## Featured Projects

### 🛍️ Zentrais Marketplace — *Lead Frontend Engineer*
**Problem:** The marketplace needed fast, reliable search and secure auth flows to scale,
but the frontend had no shared component system — leading to inconsistent UI and slow
feature delivery.

**Contributions:** Led frontend architecture for marketplace search and authentication ·
Built a reusable component library adopted team-wide · Integrated REST APIs with RTK
Query · Standardized UI patterns across the marketplace frontend

**Outcome:** Faster feature delivery across teams via shared components, and a more
reliable search and authentication experience for marketplace users.

`React` `Next.js` `TypeScript` `Redux Toolkit` `RTK Query` `Tailwind CSS`
— *Private repository (client work)*

### 💳 C-Hub — *Enterprise Fintech Dashboard*
**Problem:** Finance teams needed a central dashboard to process invoices, track
payments, and view analytics instead of relying on spreadsheets and manual reconciliation.

**Contributions:** Built invoice processing and payment tracking interfaces · Developed
analytics visualizations for finance teams · Implemented role-based access controls ·
Integrated REST APIs secured with JWT

**Outcome:** Replaced manual, spreadsheet-driven finance workflows with a single
dashboard, reducing reconciliation time and errors.

`React` `Node.js` `Express` `PostgreSQL` `REST APIs` `JWT`
— *Private repository (client work)*

### 🚚 Logistics Platform — *Full-Stack Developer*
**Problem:** Logistics operations needed real-time shipment visibility and a single
place to manage orders, payments, and scheduling.

**Contributions:** Built real-time shipment tracking UI · Developed role-based
operations dashboards · Implemented order and payment workflows · Integrated Socket.io
for automated notifications

**Outcome:** Gave operations staff real-time shipment visibility and cut manual
coordination across dispatch and support.

`React` `Node.js` `Express` `MongoDB` `Socket.io`
— *Private repository (client work)*

### 🧑‍💼 SmartStart Marketplace — *Backend Engineer (Laravel)*
**Problem:** An Upwork-style marketplace needed distinct, secure experiences for
employers, freelancers, and admins, plus messaging and job search, without fragmenting
the codebase.

**Contributions:** Built RBAC-secured REST APIs for a multi-role marketplace ·
Implemented multi-role authentication (employer, freelancer, admin) · Built in-app
messaging backend · Implemented job search and filtering logic · Designed
milestone-based escrow payment system

**Outcome:** Enabled three distinct user roles to operate securely on one platform, with
milestone-based escrow payments and reliable messaging.

`Laravel` `MySQL` `REST APIs` `RBAC` `React`
— *Private repository (client work)*
